05常见问题

不买 Pro 能用全部功能吗？▾
能。所有串流、触控、陀螺仪、压枪、ADS 灵敏度等功能永久免费。Pro 只增加云端服务与优先支持，不锁任何玩法功能。
为什么不直接卖 App？▾
Moonlight 采用 GPLv3 开源协议，fork 版本必须同样开源，不能对开源功能本身收费。因此我们采用"功能全免费 + 云端服务/优先支持收费"的合规模式。
需要哪些前置条件？▾
PC 端安装 Sunshine 服务（需 SYSTEM 权限+global_input=1 以支持反作弊游戏输入注入）；平板端安装本 App；建议 USB 直连以获得最低延迟。
支持其他游戏吗？▾
支持。当前针对《三角洲行动》深度调优，按键方案与陀螺仪参数可自定义，适用于任何 PC 射击/动作游戏。Pro 用户可从模板库一键下载热门游戏预设。
陀螺仪手感怎么调？▾
长按右下角"设置"按钮 600ms 进入陀螺仪设置页。首次使用务必先校准（静止 2 秒）。建议从默认值开始，逐步微调灵敏度与软平滑阈值。
